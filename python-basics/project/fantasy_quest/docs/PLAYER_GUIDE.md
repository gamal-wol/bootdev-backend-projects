## Getting Started

Navigate to: `d:\GitHub\bootdev-backend-projects\python-basics\project\fantasy_quest`
Run: `python main.py`

**First Steps:**
1. Equip Rusty Sword and Leather Armor (Menu Option 3)
2. Check Quest Log for Goblin Slayer quest (Menu Option 4)
3. Start Exploring to fight enemies (Menu Option 1)

## Quick Reference

### Starting Stats
- HP: 100, ATK: 10, DEF: 5, Gold: 50
- With equipment: ATK 15, DEF 10

### Level Milestones
- Level 2: Buy Iron Sword (50g)
- Level 3: Start Orc Hunter quest
- Level 10: Prepare for Dragon

### Combat Priority
1. Potion if HP < 50%
2. Attack if HP is safe
3. Flee if outmatched
